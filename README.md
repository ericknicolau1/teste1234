# MeuSite

Site simples e responsivo com 5 páginas, desenvolvido com HTML, CSS e JavaScript.

## Páginas

| Página | Arquivo | Descrição |
|--------|---------|-----------|
| Início | `index.html` | Página principal com cards de destaques |
| Sobre | `sobre.html` | História da equipe e membros |
| Serviços | `servicos.html` | Lista de serviços oferecidos |
| Galeria | `galeria.html` | Grid de projetos interativos |
| Contato | `contato.html` | Formulário de contato com validação |

## Tecnologias

- **HTML5** — Estrutura semântica
- **CSS3** — Layout responsivo com Grid e Flexbox
- **JavaScript** — Interatividade (menu mobile, modal, formulário)

## Funcionalidades

- Design responsivo (mobile e desktop)
- Menu hamburger em telas pequenas
- Cards com animação de hover
- Galeria interativa com modal
- Formulário de contato com feedback visual
- Navegação com destaque da página ativa

## Como Usar

Abra o arquivo `index.html` no navegador.
